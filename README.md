# TrackTable

**TrackTable** – это инструмент аудита изменений для Excel-файлов.  
Он позволяет вести детальный журнал того, кто, когда, где и что изменил в таблице, не требуя развёртывания корпоративных систем учёта.

Проект создан для компаний и команд, которые хотят:

- работать в привычном Excel;
- отслеживать изменения без сложной инфраструктуры;
- получить готовое решение «под ключ» или собрать свой EXE.

## Возможности

- ✅ **Аудит изменений** – фиксируется каждое изменение ячеек, добавление/удаление листов, переименование столбцов.
- ✅ **Журнал действий** – записывается время изменения, имя пользователя Windows, лист, номер строки/столбца, заголовки, старое и новое значение.
- ✅ **Резервное хранение** – копия базы аудита автоматически сохраняется в указанную папку (на случай удаления основного файла).
- ✅ **Работа без сервера** – все данные хранятся в SQLite рядом с Excel-файлом и в резервной папке.
- ✅ **Простота для конечного пользователя** – готовый EXE, который после нажатия кнопки открывает Excel, ждёт закрытия и записывает изменения.
- ✅ **Многопользовательский режим** – встроенная блокировка файла и предложение открыть копию, если оригинал уже редактируется.
- ✅ **Конфигуратор** – графическое приложение, которое за 2 минуты создаёт EXE-аудитор под конкретный файл.

## Как это работает

1. **Администратор** запускает **TrackTable Configurator** (конфигуратор), указывает:
   - путь к Excel-файлу;
   - папку для резервной копии базы аудита;
   - пароль администратора (для создания эталона);
   - иконку (стандартную или свою).
2. Конфигуратор генерирует Python-скрипт и компилирует его в **EXE-аудитор** (через PyInstaller).
3. **Пользователи** запускают полученный EXE-файл:
   - **«Открыть оригинал»** – блокировка файла, запуск Excel, ожидание закрытия, запись изменений в базу.
   - **«Получить копию»** – создание копии Excel во временной папке, открытие папки в проводнике.
4. Администратор может в любой момент нажать **«Создать базу (админ)»** (после ввода пароля), чтобы зафиксировать начальное состояние (эталон) или пересоздать его.

## Установка и использование

### Для конечных пользователей (готовый EXE)

1. Скачайте с [релизов](https://github.com/genpol/TrackTable/releases) файл `TrackTableConfigurator.exe`.
2. Запустите его.
3. Следуйте шагам конфигуратора:
   - выберите Excel-файл (`.xlsx` / `.xlsm`);
   - выберите папку для резервной копии базы аудита;
   - укажите пароль администратора;
   - при желании выберите свою иконку.
4. Нажмите **«Создать .EXE»** – через минуту в папке `dist` появится готовый аудитор.
5. Раздайте этот аудитор сотрудникам.

### Для разработчиков (сборка из исходников)

Если вы хотите собрать конфигуратор самостоятельно или изменить шаблоны:
